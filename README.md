# GTD TV Studio — Privacy Policy

This repository hosts the privacy policy for the apps published by **GTD TV Studio**:

- [GTD Launcher](#) — Android TV custom launcher
- [GTD Plugin for Emby](#) — Wallpapers, screensaver, channels and football overlays for the launcher
- [GTD Setup](#) — Mobile companion app for QR-pairing the launcher

The policy is served via GitHub Pages:

- 🇬🇧 English: <https://gokutd.github.io/gtd-policy/>
- 🇪🇸 Español: <https://gokutd.github.io/gtd-policy/es.html>

## Summary

We do not collect, transmit or sell any personal data. Everything you configure in the apps stays on your device. Optional integrations (Google Drive backup, Emby, TMDB, football APIs) only activate when you explicitly enable them, and you control the credentials.
